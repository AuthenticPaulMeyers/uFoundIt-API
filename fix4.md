
In messages:

Apply a working Chat system using socket IO. Using the Chat template make sure users are able to get real time messages with status to show the delivery of the message and when the user is active.



Users can view the profile of the user they are chatting with to see their stats and whether they are a verified user or not. They can call the other user from their profile where they can click on the call button.



Users can search and toggle chats. Get notified on the number of messages unread on the chat they haven't viewed.



Users can search a list of chats.
